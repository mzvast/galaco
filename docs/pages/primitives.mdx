# Box

项目中所有元素都基于 Box。Box 的 props 可以传入任意其他元素。
Box 包含了一个`<div>`元素和 reset 样式。支持传入 CSSProps，支持 palette 中的 color。

`import { Box } from 'galaco'`

<Playground>
    <Box>This is a box</Box>
</Playground>

## StyleSheetManager Stylis 插件 整合:

<Playground>
   <StylisBox></StylisBox>
    
</Playground>

## CSSProps 整合:

<Playground>
    <Box backgroundColor="primary" color="white" padding="major-3">
        This is a box
    </Box>
</Playground>

## Palette 整合

Any CSS color attribute (e.g. `color`, `border-color`, etc) can take a [color from the palette](/palette#theming).

<Playground>
    <Box backgroundColor="primaryTint" color="primary">
        This is a box
    </Box>
</Playground>
