# galaco 🌈

## What's this for

提供了构建 react app 的 UI 相关的功能组件，帮助开发者快速构建复杂界面。
It's a toolset based on styled-components ecosystem to build React UI components.

## Install

`npm i --save galaco`

### develop

```
yarn
npm run docz:dev // 运行交互式文档server
npm run build // 编译产出到dist目录，配置详见rollup.config.js
```
