# Palette

Palette是全局调色板

## 使用palette

我们导出了palette工具函数，可以用于取得颜色

```tsx
import {Box, palette, styled} from '@mzvast/galaco';

const PrimaryBox = styled(Box)`
    color: ${palette('primary')};
`;

<PrimaryBox> I am a primary box</PrimaryBox>;
```
